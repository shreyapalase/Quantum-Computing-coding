# Quantum Data Encoding
In this folder contain material for amplitude encoding for given classical dataset.

In this we will learn:
- What is **Quantum Data Encoding**
- Concept of **Amplitude Encoding**
- Encoding a **4D classical vector** using 2 qubits
- Manual state preparation using rotation gates
- Visualization and verfication using simulation
---

## Contents:
1. **day25_QuantumDataEncoding.ipynb** 
this hands-on jupiter practice notebook which represent demonstration of working Quantum Data Encoding in Quantum Computing.

2. **day25_notes.md**
   this file include detailed theory notes on Quantum Data Encoding with well explained our implementation for more deep understanding.
